Info

MP4 files can be tested in Tizen Emulator. Other formats need TV.

This app plays content with subtitle,

using the WebVTT format and the track element.

[Available Subtitle] - English, Korean

Of course, other languages can be added.

Operation

Use remote controller to operate on video player for basic control.

- [Available keys] :

	[PLAY] - play the video
	[PAUSE] - pause the video
	[STOP] - stop the video
	[ENTER] - change the screen size
	[RED] - show or hide the subtitle
	[RETURN] - hide the application

And moreover you can control play & subtitle by using the mouse.
